Serveur Gateway:

Le serveur Gateway a pour but de distribuer Internet sur toute les VM de tous les ordinateurs de notre groupe.
Une fois les VM crées et les configuration faite, on s'est tous mis en mode bridge ce qui nous a permis de nous connecter entre nous.
